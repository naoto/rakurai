# Rakurai

Rakurai is basic auth pass server

## Installation

And then execute:

    $ bundle install 

## Configuration

Write config.yaml

```yaml
---
base_uri: "http://localhost"
username: "user"
password: "pass"
port: 9000
```

## Usage

```shell
$ bundle exec bin/rakurai
```
